Additional HWHC projects

Document manager

Description

Document manager is used by internal staff to view and create reports.

Document manager is a singular code base that can be configured by teams. We completed accessibility fixes on the PECS and PPC document manager systems. They're 2 different systems, but come from the same code base. So we only had to complete the fixes once.

It's a legacy system that is not updated often. The accessibility fixes were the first update for years.

The code is hosted on SVN.

To access the Document manager test system you need to be on the Global VPN.

Languages

Java 8 HTML5 CSS3 JSP

PECS Web

Description

PECs web is an application used by internal staff to process prescription exemptions.

You need tomcat 8.5, access to the PECS db and a docker image to run this locally

Local url will be: http://localhost:9095/pecs-web/

Languages

Java 8 HTML5 CSS3 Javascript JSP

LIS Form Ordering

Toolkit version: Elements v1.0.22954

Description

Users who want to apply for the NHS Low income scheme can order paper forms to be delivered using LIS Form Ordering.

Notes

LIS form ordering is hosted on SVN and requires a Jboss server to run locally.

When deploying frontend changes to LIS form ordering remember to get the extract tested. The extract sends user data to the required department so they can complete the users request for printed LIS forms.

Local url will be: http://localhost:8080/LISWebApp/adminPage.do
